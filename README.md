# Katoolin-tr
Katoolin Aracının Türkçe'ye Çevirilmiş Halidir.

# Kurulum
- sudo su
- git clone https://github.com/kernelblog/katoolin-tr.git && cp katoolin/katoolin.py /usr/bin/katoolin
- chmod +x /usr/bin/katoolin
- sudo katoolin
